# Changelog

## 1.3.1 - 2026-05-17

- 计算结果新增大小写切换：每个算法行右侧提供独立切换按钮，头部新增「全部小写」「全部大写」批量入口。
- 设置页新增「默认大小写」选项，决定后续新计算结果的初始显示，并持久化保存。
- 复制行为遵循「所见即所复制」：单值复制、卡片复制全部、复制全部结果均按当前显示大小写输出。
- 切换为纯渲染层操作，不触发任何重新计算，未完成任务上的切换按钮保留焦点可达性。

## 1.2.1 - 2026-05-13

- 优化暗黑模式下输入框聚焦态、批量进度面板和统计标签的视觉一致性。

## 1.2.0 - 2026-05-13

- 增加可选额外算法：CRC32、SHA384、SHA512。
- 设置页支持开关额外算法，默认仍只启用 MD5、SHA1、SHA256。
- 文件计算保持单次流式读取，同时计算所有已启用算法。
- 复制单项和复制全部结果会包含当前任务启用的算法。

## 1.1.0 - 2026-05-13

- 优化整体 UI 风格与紧凑布局。
- 增加批量任务完成后的页面内轻提示。
- 设置插件窗口高度，改善默认打开体验。
- 修复计算中再次从 ZTools 文件入口进入会覆盖当前任务的问题。
- 修复长路径显示方向异常的问题。
- 增加一键打包 zip 的 npm 脚本。

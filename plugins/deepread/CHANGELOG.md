# Changelog

## v1.1.3

- 优化多书源在线搜索流程，单个书源遇到验证码或访问验证时不再阻断其他书源结果，并支持按书源展示验证、失败和单源重试状态。
- 增强验证码处理，支持从通用 `getVerificationCode` 规则推断图片验证码请求，验证码输入后仅重试对应书源。
- 修复验证页打开兼容性，新增本地验证跳转页，并优先保留 HTTP/HTTPS 请求的 Cookie Jar 行为，提升站点验证后重试成功率。
- 优化书源设置，新增剪切板导入，改为链接导入与 JSON 导入分离，并将 JSON 输入移动到独立弹窗。
- 调整在线书籍操作文案，将搜索结果和详情页的“导入”统一为“加入书架”。
- 将目录改为与书内搜索一致的弹窗，并精简顶部工具栏的目录、书内搜索和显示隐藏按钮。
- 移除书源仓库推荐入口，新增首次使用免责声明弹窗，用户同意后才能继续使用插件。

## v1.1.2

- 将书内搜索从侧栏页面改为弹窗，支持打开后自动聚焦、跳转后关闭和 `Esc` 关闭。
- 将找书页面调整为独立在线搜索页面，保留验证、结果详情和导入流程，并补充无书源与空结果状态。
- 将书源设置拆分为独立页面，集中管理书源链接导入、JSON 编辑、启停和删除操作。
- 优化找书、书源设置和书内搜索弹窗的响应式布局与空态样式。

## v1.1.1

- 调整 ZTools 插件入口命令，仅保留 DeepRead 主入口，并将入口关键词补充为“阅读”。
- 增强书源规则解析，支持未显式标记的 JS 规则，并在配置列表规则时避免误用搜索结果回退解析。
- 优化搜索验证页识别，空结果时优先使用最后一次规则输入内容判断访问验证状态。
- 修正预加载网络服务的 Google 跳转域名判断、Cookie 过期处理、连接中止错误捕获和文件读取调用上下文。

## v1.1.0

- 新增在线书源能力，支持粘贴或通过链接导入阅读 APP 常见 JSON 书源，并对书源进行保存、启停和删除。
- 新增搜书面板，支持跨启用书源搜索在线书籍、查看详情和目录预览，并从搜索结果或详情页导入书架。
- 新增书源规则解析器，支持常见搜索、书籍信息、目录和正文规则，以及 CSS / class / tag / XPath / JSON / 正则 / JS 等解析方式。
- 支持在线书籍缓存章节目录和当前章节正文，书架按本地书籍、在线书籍分区展示，并在书籍卡片上展示来源、作者、最新章节和章节进度。
- 支持在线章节阅读流转，页尾自动加载下一章、页首加载上一章，目录跳章会实时抓取目标章节正文。
- 增强验证码和访问验证处理，支持图片验证码弹窗、浏览器验证页、验证后 Cookie 保存、验证页内容回读和重试搜索/导入。
- 增强预加载网络请求服务，支持 HTTP/Electron net 请求、重定向处理、Cookie Jar、压缩响应解码、图片 Data URL 读取和 UTF-8 JSON/中文编码兼容。
- 优化找书、搜索结果、书籍详情、验证码弹窗和详情页章节目录样式，提升在线书籍导入流程的状态提示。
- 新增 ZTools “搜书/找书”入口命令，并补充 README 在线书源使用说明。

## v1.0.2

- 修正底栏阅读窗口显示元信息时未预留高度，可能导致正文可读行数估算偏多的问题。
- 优化底栏阅读窗口元信息布局，在高度不足时自动隐藏元信息，避免挤压正文区域。
- 修正鼠标移出隐藏逻辑，使其不再依赖低调模式开关。
- 微调控制栏间距，并将透明度设置文案调整为“窗口透明度”。

## v1.0.1

- 允许将阅读器透明度设置为 `0%`。
- 修正底栏阅读窗口在透明度为 `0` 时错误回退到默认透明度的问题。
- 优化整本书分页起点计算，减少重排时的字符串和数组分配，降低大文件阅读时的卡顿风险。
- 调整 `ztools` 存储读取调用写法，避免可选链后直接跟泛型参数带来的 TypeScript 兼容性问题。
- 优化设置持久化监听，移除不必要的浅拷贝对象创建。
- 避免底栏阅读窗口已存在时重复执行状态同步，减少高频更新下的额外 IPC 开销。

## v1.0.0

- 初始发布 DeepRead 深读。
- 支持本地 `.txt` / `.md` 小说导入、书架管理和阅读进度保存。
- 支持分页阅读、目录解析、全文搜索、自动翻页和滚轮/键盘翻页。
- 支持低调外观、快速隐藏、鼠标移出隐藏和独立底栏阅读窗口。
